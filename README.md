# OWL-OS

## VERY EARLY DEVELOPMENT

## Install

```sh
$ git clone http://github.com/neutrino2211/OWL-OS
    ...
    ...

$ cd OWL-OS
$ pip3 install -r requirments.txt
$ python3 install.py <options>
```

## Run
```sh
$ chmod +x ./run.sh
$ ./run.sh
```
